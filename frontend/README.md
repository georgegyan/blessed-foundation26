# Blessed Foundation Frontend

React frontend for Blessed Foundation NGO application.

## Tech Stack
- React 18 with Vite
- Tailwind CSS v4
- React Router v6
- Axios for API calls

## Features (Phase 1)
- Home page with hero section
- Join Us form with validation
- Donation form with predefined amounts
- Success/error messaging
- Loading states
- Responsive design
